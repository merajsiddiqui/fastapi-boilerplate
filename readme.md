# PhysioX API

PhysioX API development in python with the help of fastAPI

## Setup

```bash
python --version

>> Python 3.8.9

```

1. ` virtualenv venv`
2. `source venv/bin/activate`
3. `pip install -r requirements.txt`
4. `cp example.app.in`

> ### Running the application
> `uvicorn main:app --reload`
